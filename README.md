# Digital Logic Gates using Logisim

## Objective

This project aims to design, simulate, and verify the operation of basic digital logic gates and a Half Adder using Logisim. It helps in understanding the fundamentals of digital electronics and the behavior of combinational logic circuits.

## Project Contents

- AND Gate
- OR Gate
- NOT Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
- Half Adder

## Tool Used

- Logisim

## Description

Each circuit has been created and tested individually in Logisim to ensure correct functionality. This project was completed as part of an internship task to strengthen practical knowledge of digital circuit design.
